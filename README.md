# Travel X More — client demo

A complete React + Vite travel-booking prototype for Travel X More, Jodhpur. It includes package discovery, simulated authentication, traveler details, simulated checkout, and a printable booking voucher.

## Run locally

```bash
npm install
npm run dev
```

Use `npm run build` for a production build.

Authentication, enquiries, contact submissions, and payment are simulated entirely in the browser. No payment data is transmitted and no backend is connected.
